# Kindergarten Snake Game

Welcome to the Kindergarten Snake Game! This engaging and colorful game is designed to enhance enthusiasm and learning for kindergarten students through fun gameplay and vibrant visuals.

## Project Overview

The Kindergarten Snake Game is a simple yet captivating game where players control a friendly snake that moves around the screen, collecting various shapes and colors. The game encourages hand-eye coordination, shape recognition, and color identification in a playful environment.

## Features

- **Engaging Gameplay**: Players guide the snake to collect items while avoiding obstacles.
- **Colorful Visuals**: Bright and cheerful graphics designed to attract and hold the attention of young children.
- **Simple Controls**: Easy-to-use controls suitable for kindergarten students.
- **Educational Elements**: Incorporates learning through shape and color recognition.

## Installation

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/kindergarten-snake-game.git
   ```
2. Navigate to the project directory:
   ```
   cd kindergarten-snake-game
   ```
3. Install the required packages:
   ```
   pip install -r requirements.txt
   ```

## Usage

To start the game, run the following command:
```
python src/main.py
```
